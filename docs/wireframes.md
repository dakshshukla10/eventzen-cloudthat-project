# EventZen - UI Wireframes

## Design System

- **Theme**: Dark mode
- **Primary Color**: Indigo (#6366f1) with purple gradient accents
- **Font**: Inter (Google Fonts)
- **Layout**: Max-width 1200px centered container
- **Cards**: Rounded corners (12px), subtle borders, hover elevation
